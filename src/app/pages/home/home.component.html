Resumen

Este sistema de demostración reproduce un proceso logístico real utilizado para transformar reportes de carga generados por un sistema interno en un formato de Excel estandarizado listo para ser enviado.

La herramienta analiza el archivo de origen, extrae la información relevante y agrupa los datos por finca exportadora y variedad de flor, generando una versión organizada del reporte.

Como resultado, el sistema presenta:

  • Los datos adicionales requeridos por el reporte final, que pueden ser modificados antes de generar el archivo.
  • Una tabla con la información procesada, donde las filas pueden reordenarse según sea necesario.

Finalmente, el usuario puede descargar un archivo de Excel con el reporte formateado y listo para enviar.

Instrucciones

  1. Descargue el archivo de reporte de ejemplo.
  2. (Opcional) Modifique los datos del archivo para probar distintos escenarios.
  3. Cargue el archivo en el sistema utilizando el selector de archivos.
  4. El sistema procesará automáticamente la información del reporte.
  5. Revise y edite los datos adicionales si es necesario.
  6. Reordene las filas de la tabla según sus necesidades.
  7. Descargue el archivo final en formato Excel listo para enviar.

Aviso

Este sistema es una demostración basada en un proceso logístico real. Todos los datos utilizados en el archivo de ejemplo han sido anonimizados o generados con fines de demostración.

La aplicación funciona completamente del lado del cliente (en el navegador). Ningún archivo cargado ni información procesada es enviado a servidores externos ni almacenado por el sistema.

El uso de este sistema y de cualquier archivo generado a partir del mismo es responsabilidad exclusiva del usuario.
